# GIS Technical Assesment
## Author: Qanita Baqibillah
## Date: 06/09/2026
## Study Area: Nairobi, Kenya 

This project is an automated GIS analysis and data science framework created for a technical assesment. 

This project performs automated geospatial analysis using python. It is fully autommated using GeoPandas and open source spatial datasets. It processes administrative boundaries, infrastructure, and land use data to create district level maps, risk metrcsi, as well as spatial relations.

Datasets used:
- Administrative boundaries GeoJSON from OpenStreetMap
- Infrastructure data of hospitals and fire Stations from OpenStreetMap
- Land use polygons from OpenStreetMap landuse data

Tools used:
- Python 3.12
- GeoPandas
- Pandas
- Shapely
- Matplotlib
- NumPy
